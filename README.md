# Hi there, I'm Vedang Barman

## About Me
- I am currently working on **GIT PR PREDICTOR ** (A machine learning early-warning system that forecasts GitHub Pull Request merge timelines using multi-checkpoint classification. It analyzes point-in-time repository activity and behavioral signals to help maintainers proactively identify and triage code review bottlenecks.) 
- I am passionate about data, machine learning, and building efficient systems.
